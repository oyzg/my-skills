## 概览
本文档提供 AI & 科技类文章的风格分析框架和应用指南。通过系统化的维度拆解，帮助智能体学习特定风格，并在写作中自然应用，避免机械模仿。

## 风格分析维度

### 1. 句式特点
**分析要点**：
- 长短句比例
  * 长句：>30 字（表达复杂观点）
  * 短句：<15 字（制造节奏、强调）
  * 黄金比例：长句 40% + 中句 40% + 短句 20%

- 句型结构
  * 陈述句：陈述事实和观点
  * 设问句：引发思考
  * 反问句：强调观点
  * 感叹句：表达情绪

- 逻辑连接词使用
  * 过度使用："首先、其次、再次、综上所述"（AI 感明显）
  * 自然表达："但"、"不过"、"有意思的是"（更自然）

**示例对比**：
```
❌ AI 感明显：
首先，GPT-5 的发布具有重要意义。其次，它将改变内容创作方式。最后，企业需要提前布局。

✅ 自然表达：
GPT-5 的发布意义重大，它正在重塑内容创作。有意思的是，很多企业已经开始提前布局了。
```

### 2. 词汇选择
**分析要点**：
- 专业术语 vs 通俗表达
  * 专业密度：多少专业术语 / 千字
  * 解释方式：是否用比喻/类比解释
  * 目标读者：面向专业人士还是普通读者

- 语气态度
  * 严肃专业：使用客观、中性的词汇
  * 轻松幽默：使用网络用语、梗、调侃
  * 热情推荐：使用积极、鼓励的词汇

- 常用高频词
  * 某些媒体喜欢用特定词汇（如"颠覆"、"革命"、"天花板"）
  * 统计参考文章的高频词汇表

**示例对比**：
```
❌ 过于学术：
该模型采用了 Transformer 架构，通过多头注意力机制捕捉长距离依赖关系，在自然语言处理任务中表现优异。

✅ 通俗易懂：
简单说，这个模型就像个超强大的"理解者"，能一下子记住很多上下文，所以写东西特别厉害。
```

### 3. 修辞手法
**分析要点**：
- 比喻/类比
  * 使用频率
  * 比喻类型（生活类比、历史类比、技术类比）
  * 比喻的创意度

- 对比
  * 时间对比（过去 vs 现在）
  * 数据对比（A vs B）
  * 正反对比（优点 vs 缺点）

- 设问/反问
  * 使用位置（开头、中段、结尾）
  * 问题类型（开放性 vs 引导性）

- 排比/对仗
  * 使用场景（强调、总结）
  * 排比数量（2 句/3 句/4 句）

**示例**：
```
比喻示例：
- "ChatGPT 就像给全世界配了个超级助理"
- "这个技术就像给 AI 装上了眼睛"
- "数据是新时代的石油，但算法才是提炼石油的炼油厂"

对比示例：
- "3 年前没人看好，现在估值 100 亿"
- "用 AI 写文章，速度是人工的 10 倍，质量也能达到 8 成"
```

### 4. 段落节奏
**分析要点**：
- 段落长度
  * 短段落：1-2 句（制造节奏、强调）
  * 中段落：3-5 句（正常叙述）
  * 长段落：6+ 句（详细展开）

- 段落功能
  * 开篇段落：吸引注意力
  * 过渡段落：承接上下文
  * 核心段落：阐述观点/案例
  * 总结段落：提炼要点

- 段落过渡
  * 自然过渡：无过渡词，依靠逻辑衔接
  * 显性过渡：使用"但"、"不过"、"更有意思的是"
  * 数字标号：1、2、3（适合总结要点）

**示例**：
```
短段落效果：
GPT-5 来了。
这次真的不一样。
它不仅写文章更快，而且能理解你的意图，更像是一个真正的"助理"。

长段落效果：
从技术角度看，GPT-5 的突破主要体现在三个方面。首先是参数规模的持续扩大，达到了惊人的万亿级别，这让模型能够处理更复杂的上下文。其次是多模态能力的整合，文本、图像、音频可以统一处理，大大扩展了应用场景。最后是推理能力的提升，不再是简单的预测下一个词，而是能够进行真正的逻辑推理。
```

### 5. 情绪表达
**分析要点**：
- 情绪基调
  * 理性客观：平铺直叙，观点中立
  * 激情澎湃：用感叹号、强烈词汇
  * 幽默调侃：轻松、讽刺、自嘲

- 情绪曲线
  * 开头：制造悬念/冲突（高情绪）
  * 中段：理性分析（中情绪）
  * 结尾：升华/行动召唤（高情绪）

- 人设感
  * 专业人士：权威、严谨
  * 行业老司机：轻松、有洞见
  * 新手小白：好奇、学习心态

## 常见风格类型

### 1. 36氪风格
**特点**：
- 短段落为主（1-2 句）
- 频繁使用数据和案例
- 节奏快，信息密度高
- 标题党但有干货

**句式示例**：
```
OpenAI 又搞事情了。
这次是 GPT-5。
参数量是上一代的 10 倍。
```

**词汇特征**：
- 频繁使用："颠覆"、"革命"、"天花板"
- 大量数据支撑
- 行业黑话适度使用

### 2. 极客公园风格
**特点**：
- 技术深度 + 通俗解释
- 喜欢用比喻和类比
- 语气略带调侃和幽默
- 重视产品细节

**句式示例**：
```
简单说，这个技术就像给 AI 装上了"眼睛"。
之前 AI 只能"听"，现在能"看"了。
有意思的是，这个"眼睛"还特别厉害。
```

**词汇特征**：
- 适度使用网络用语
- 技术解释喜欢用生活类比
- 语气轻松，但内容专业

### 3. 虎嗅风格
**特点**：
- 深度分析，逻辑严密
- 喜欢反直觉观点
- 引用权威资料
- 语气理性客观

**句式示例**：
```
表面上看，这是一场技术革命。
但深层逻辑是商业模式的重构。
传统企业的护城河正在崩塌。
```

**词汇特征**：
- 使用经济学术语
- 引用数据和报告
- 观点犀利，逻辑清晰

## 去 AI 感技巧

### 1. 消除机械句式
**问题**：
- 过度使用"首先、其次、再次、最后"
- 三段式结构太明显
- 逻辑连接词太生硬

**解决**：
```
❌ AI 感明显：
首先，GPT-5 的发布具有重要意义。其次，它将改变内容创作方式。最后，企业需要提前布局。

✅ 自然表达：
GPT-5 的发布意义重大，它正在重塑内容创作。更有意思的是，很多企业已经开始提前布局了。

❌ AI 感明显：
优点是速度快，缺点是质量不稳定。

✅ 自然表达：
速度确实很快，但质量还是有点不稳定。
```

### 2. 注入情绪和观点
**问题**：
- 过于客观，缺乏个人观点
- 像"新闻报道"而不是"文章"
- 缺乏情绪起伏

**解决**：
```
❌ AI 感明显：
GPT-5 发布了，参数量很大，功能很强。

✅ 自然表达：
GPT-5 终于来了！说实话，这次真的很惊喜，参数量直接翻倍，但更厉害的是推理能力。

❌ AI 感明显：
这个技术有优点，也有缺点。

✅ 自然表达：
我个人觉得，这个技术确实厉害，但也有点让人担心。毕竟，AI 越来越像人了。
```

### 3. 使用口语化表达
**问题**：
- 用词过于正式
- 句式过于规整
- 缺乏对话感

**解决**：
```
❌ AI 感明显：
因此，我们需要关注这个问题。

✅ 自然表达：
所以，这个问题真的值得重视。

❌ AI 感明显：
该模型的表现令人满意。

✅ 自然表达：
说实话，这个模型的表现还挺让人惊喜的。
```

### 4. 打破三段式结构
**问题**：
- 开头、中段、结尾界限太明显
- 每段都差不多长
- 缺乏节奏变化

**解决**：
- 用短段落制造节奏
- 用问句打破节奏
- 用数据或案例插入

```
短段落示例：
GPT-5 来了。
这次真的不一样。

长段落示例：
从技术角度看，这次升级主要体现在三个方面。首先是...其次是...最后是...

问句示例：
这会不会是 AI 时代的"iPhone 时刻"？
我觉得很有可能。
```

## 风格应用方法

### 1. 风格学习流程
1. 读取 2-3 篇参考文章
2. 分析 5 个维度（句式、词汇、修辞、段落、情绪）
3. 提取关键特征（3-5 条）
4. 生成风格报告

### 2. 风格应用流程
1. 生成初稿（不关注风格，先完成内容）
2. 逐句对照风格特征调整
3. 多轮润色（每轮聚焦 1 个维度）
4. 最终检查（整体风格一致性）

### 3. 风格验证标准
- 句式：长短句比例合理
- 词汇：符合目标读者习惯
- 修辞：适度使用比喻/对比
- 段落：有节奏变化
- 情绪：有观点和态度

### 4. 常见问题
- ❌ 机械模仿：过度复制具体句式
- ✅ 理解精髓：掌握风格背后的逻辑
- ❌ 风格冲突：一篇文章多种风格混搭
- ✅ 风格统一：保持整体一致性
- ❌ 过度风格化：为了风格牺牲内容
- ✅ 风格服务内容：风格是锦上添花
